集合-离散数学

集合与元素

  • 集合是人们能够明确区分的一些对象(客体)构成的一个整体。

    集合通常用大写英文字母表示，其中用：

    N表示自然数集合(含0)，R表示实数集合，Q表示有理数集合，Z表示整数集合。

  • 元素：集合里含有的对象称为该集合的元素。

    通常用小写英文字母表示元素。

如果a是集合A中的元素，就写成a$\in$A，读作a属于A。

如果b不是集合A中的元素，就写成b$\notin$A，读作b不属于A。

集合的表示方法：

  1. 枚举法：把集合中的元素全部列举出来，用 { } 括起来， 元素之间用逗号分开。

  2. 抽象法:用 谓词 概括集合中元素的属性。
    其描述形式为 S = { x | P(x) }。

    抽象原则任给一个性质 P，就确定 了一个集合A , A的元素恰好是具有性质 P 的对象，即A = { x | P(x) }。

    抽象原则的限制：1. 谓词 P(x) 要明确清楚。

    1. 不能取 P(x) 为 x $\notin$ x 这样的谓词来定义集合，否则就会产生悖论(罗素悖论，B.Russell）。

    关于集合的直观描述不能作为集合的严格定义。

    若C = { S | S是集合且 S $\notin$ S }，则C*不是集合。

  3. 归纳定义法：

    1) 基本项

    ​ 非空集 **$S_0 \in A$；

    2) 归纳项

    ​ 一组规则，从A中元素出发，依据这些规则所获得的元素仍然是A中的元素；

    3) 极小化：

    ​ 保证A中每个元素都可通过有限次使用1) 或2)来获得。

    如果集合S$\subseteq$A也满足 1) 和 2)，则A$\subseteq$S。

    （如果集合S$\subseteq$A也满足 1) 和 2)，则S=A）

    • 极小化保证A 是满足 1) 和 2) 的最小集合。

集合间的相等和包含关系

  • 定义5.2 (集合相等)（外延性公理）：设 A, B为任意两集合，若A和B含有相同的元素，则称A和B相等，记作：A=B。

    $A=B\iff\forall x(x\in A\leftrightarrow x\in B)$，或

    $A=B\iff\forall x(x\in A\rightarrow x\in B)\bigwedge \forall x(x\in B\rightarrow x\in A)$ 。

    集合与其元素排列次序、重复出现次数无关。

集合的包含关系：

  • 定义5.3 (子集或包含)：若集合A的每一个元素都是集合 B 的元素，则称 A是B的子集，也称A包含于B或B包含A。 记作：A $\subseteq$B 或 B $\supseteq$ A

    $A\subseteq B \iff \forall x(x\in A \rightarrow x \in B)$.

    由定义5.3 可知：对任意集合A有: A $\subseteq $ A

  • 定义5.4（真子集或真包含）：设 A, B是任意集合，若 A $\subseteq$ B 且 A ≠ B，则称 A为B的真子集，也称A真包含于 B，或 B真包含A。 记作：A $\subset$ B 或 B $\supset$ A。

    $A\sub B\iff A \subseteq B \bigwedge A≠B\iff\forall x(x\in A\rightarrow x\in B)\bigwedge \exist x(x\in B\bigwedge x\notin A)$

  • 关系“$\in $”和“$\subseteq $”的区别：

    ​ $\in$：构成集合A的元素 a与A是 “$\in$”关系

    ​ $\subseteq$：集合A的子集B与A是 “$\subseteq$”关系

全集与空集：

  • 定义5.5（全集U）：在对集合的研究中，如果所讨论的集合，都是某一固定集合U的子集，就称集合 U 为全集，记作：U

    $U= \lbrace x|p(x)\or \neg p(x) \rbrace$

    全集是一个相对概念，故全集U不唯一。

  • 定义 5.6（空集Æ）：不含有任何元素的集合称为空集, 记作 : $\emptyset $

    $\emptyset = \lbrace x | p(x) \and \neg p(x) \rbrace$
